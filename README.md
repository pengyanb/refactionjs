# refactionjs

Refactoring exercise for Node

To run dev server:

```
npm install
npm start
```

Browse to [http://localhost:3000](http://localhost:3000)

To run production server:

```
npm install
npm run start:production
```

Browse to [http://localhost:3000](http://localhost:3000)

To run client side tests:

```
npm install
npm run test:app
```

To run server side tests:

```
npm install
npm run test:server
```
